# COCO Dataset Analysis

This project explores and analyzes the COCO Captions dataset to understand its structure and characteristics.

## Analysis Performed
- Total number of images
- Total number of categories
- Average caption length
- Visualization of images with corresponding captions

## Technologies Used
- Python
- PyCOCOTools
- Matplotlib
- NumPy

This project demonstrates dataset understanding, statistical analysis, and multimodal visualization.
